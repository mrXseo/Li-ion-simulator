# Элементы симуляции (elements)

Конкретные компоненты, реализующие поведение симуляции. Каждый наследуется от `SimulationObject` и определяет свою логику шага.

## Файлы

### [[noise_generator.py]]

**Класс `NoiseGenerator`** — генератор гауссовского шума.

- Наследуется от `SimulationObject`.
- Параметры:
  - `sigma` — среднеквадратическое отклонение (≥ 0).
  - `bias` — систематическое смещение (постоянная составляющая).
  - `output_key` — ключ, под которым значение сохраняется в словаре результата (например, `"I_noise"`).
  - `noise_type` — пока только `"gaussian"`, но всегда можно расширить.
  - `frame_list_size` — размер очереди истории (передаётся в родительский класс).
- Метод `solve_frame()` генерирует одно случайное значение по закону `bias + N(0, sigma^2)` и сохраняет его в историю через `_push_result({output_key: value})`.
- Метод `reset_state()` — не делает ничего (нет накапливаемого состояния).
- Метод `set_parameters(sigma, bias)` — позволяет динамически менять параметры шума (без сброса истории).
- Свойство `current_noise` — возвращает последнее сгенерированное значение (или `None`, если фреймов ещё не было).

**Пример использования:**

```python
engine = SimulationEngine()
noise = NoiseGenerator(engine, sigma=0.5, bias=0.1, output_key="I_noise")
engine.step_frame()
print(noise.current_noise)   # например, 0.623
noise.set_parameters(sigma=1.0)
engine.step_frame()          # следующий шум с увеличенным разбросом
```
